## ETIQUETAS USADAS

- section: Etiqueta para dividir el contenido por secciones

```html
    <section class="container">
        <div class="resultado">
            <h2 class="cero">0</h2>
        </div>
    </section>
```
- div: Etiqueta para cambiar a otra etiqueta de tipo "en linea" a tipo "bloque"

```html
    <div class="resultado">
        <h2 class="cero">0</h2>
    </div>
```
- h2: Etiqueta de subtitulos

```html
    <h2 class="cero">0</h2>
```
